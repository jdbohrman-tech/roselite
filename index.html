    Roselite¶

    Decentralized Static Site Hosting¶

    Write once, publish everywhere. Roselite enables P2P static site hosting on the Veilid DHT network—no servers, no downtime, no censorship.

    Get Started View Source

    Why Choose Roselite?¶

    • Truly Decentralized

      Host static sites on a distributed P2P network without any centralized servers or infrastructure dependencies.

    • Privacy First

      Built on Veilid's privacy-focused DHT. Your content is encrypted and distributed across a secure network.

    • Lightning Fast

      Simple CLI commands to bundle and publish. Works with any static site generator or plain HTML files.

    • Rust Powered

      Built with modern Rust for maximum performance, memory safety, and reliability in production.

    • Version Control

      Track changes and manage different versions of your sites with built-in versioning and rollback support.

    • Global Access

      Access your sites from anywhere through the Roselite gateway—no complex DHT addresses to remember.

    How It Works¶

    Experience the future of web hosting in three simple steps:

    1. Bundle¶

    Package your static site files into a Roselite bundle with a single command.

    roselite bundle ./my-site

    2. Publish¶

    Deploy your bundle to the Veilid DHT network—no servers required.

    roselite publish site.tar.gz

    3. Share¶

    Share your gateway URL with the world. Your site is now live on the decentralized web.

    https://gateway.roselite.net/dht:...

    Traditional vs. Decentralized¶

    Traditional Hosting¶

    ❌ Server Dependencies - Relies on centralized infrastructure
    ❌ Monthly Costs - Recurring hosting fees and bandwidth charges
    ❌ Single Point of Failure - Server downtime affects your site
    ❌ Censorship Risk - Content can be removed or blocked
    ❌ Geographic Limits - Performance varies by server location

    Roselite Hosting¶

    ✅ Fully Decentralized - No central servers or infrastructure
    ✅ Zero Hosting Costs - No monthly fees or bandwidth charges
    ✅ Always Available - Distributed across peer network
    ✅ Censorship Resistant - Content persists on the network
    ✅ Global Performance - Optimized delivery worldwide
